What is this tldr

A terminal-native multiplayer space strategy game in the spirit of OGame. You play from your terminal via the tarmy CLI — every action is a slash-command. Resources accrue in real time. Multiple players share the same universe and can attack, spy on, and message each other.

Install one terminal, two commands

1. Install the CLI:

uv tool install --python 3.12 "git+https://github.com/cobanov/terminal.army.git"

2. Start playing:

tarmy

No config needed — the CLI defaults to terminal.army. On first launch it opens a browser URL, sign in there, the TUI takes over once you're authenticated. Need uv? curl -LsSf https://astral.sh/uv/install.sh | sh.
