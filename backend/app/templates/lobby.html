Install one terminal, one command

Install the sakusen CLI:

uv tool install --python 3.12 "git+https://github.com/cobanov/space-galactic-tui.git"

Point at a server and run:

export SAKUSEN_BACKEND="{% if primary_shard_url %}{{ primary_shard_url }}{% else %}https://s1.sakusen.space{% endif %}"
sakusen

Need uv? curl -LsSf https://astral.sh/uv/install.sh | sh. On first launch the CLI opens a browser URL — sign in there, the TUI takes over once you're authenticated.
